# **App Name**: CityVerse

## Core Features:

- World Environment: Display a 3D open-world environment including a city, parks, and beaches, and malls.
- Avatar Customization: Allow users to customize avatars with different faces, clothes and accessories.
- AI-Powered NPCs: Enable interactive conversations with non-player characters, using an LLM as a tool to drive dialogue.
- Entertainment Hub: Add access to virtual entertainment with short videos, concerts, and events inside the game.
- In-Game Chat: Enable basic chat functionality for players to communicate through text messages.

## Style Guidelines:

- Primary color: Saturated blue (#4287f5), reminiscent of open skies and clear days.
- Background color: Light desaturated blue (#e0e8f5) to maintain a calm feeling in a vibrant virtual space.
- Accent color: Yellow (#f5bc42), contrasting against the analogous blue hues, emphasizing important interactive elements in the game.
- Font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
